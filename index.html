Touchr

The core idea of Touchr is that many mobile web sites are prepared for touchable devices but are incompatible with new Internet Explorer 10. This package automatically mimics touch events in IE10 browser.

Usage:

<script type="text/javascript" src="js/touchr.js"></script>

Interaction:

element.addEventListener("touchstart", function(e){console.log(e)});
